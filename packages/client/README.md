# @microwallet/client

Client for µWallet. This includes the embeddable widget, React components, client SDK and a demo app testing out the widget.

## Developer guide

Install packages:

```shell
$ pnpm i
```

Run the dev server:

```shell
$ pnpm dev
```

Access the demo app at http://localhost:3001.
